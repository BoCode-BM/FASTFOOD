# FASTFOOD - Votre Restaurant en Ligne

![Aperçu du site FASTFOOD](Picture-website.png)

Bienvenue sur le dépôt GitHub du site web FASTFOOD, la vitrine numérique de notre restaurant. Ce site statique a été conçu pour présenter notre menu, nos services, et permettre aux clients de nous contacter.

## Description du Projet

Ce projet est un site web vitrine pour un restaurant de type fast-food. Il offre une expérience utilisateur simple et intuitive pour découvrir les offres du restaurant.

## Fonctionnalités

*   **Page d'Accueil (`index.html`)**: Présentation générale du restaurant, ses valeurs et ses points forts.
*   **Galerie (`galerie.html`)**: Une section dédiée aux images de nos plats et de l'ambiance du restaurant.
*   **Contact (`contact.html`)**: Un formulaire de contact et les informations pour nous joindre.
*   **Design Responsive**: Adapté pour une consultation sur différents appareils (ordinateurs, tablettes, mobiles).

## Technologies Utilisées

*   **HTML5**: Structure du contenu.
*   **CSS3**: Mise en forme et stylisation (avec des fichiers spécifiques pour chaque page et un style global).
*   **JavaScript**: Pour les interactions côté client (par exemple, la navigation, formulaire de contact simple).
*   **Polices Personnalisées**: Intégration de polices via le dossier `assets/fonts/`.
*   **Images**: Utilisation d'images provenant de Pexels pour illustrer les pages.

## Installation et Utilisation

Pour visualiser le site localement :

1.  **Clonez le dépôt** :
    ```bash
    git clone <URL_DU_DEPOT>
    ```
    (Remplacez `<URL_DU_DEPOT>` par l'URL réelle de votre dépôt)

2.  **Naviguez vers le dossier du projet** :
    ```bash
    cd FASTFOOD
    ```

3.  **Ouvrez les fichiers HTML** :
    Ouvrez simplement les fichiers `HTML/index.html`, `HTML/galerie.html` ou `HTML/contact.html` dans votre navigateur web préféré.
